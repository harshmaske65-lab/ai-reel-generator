# AI Reel Generator

A Flask-based web application for generating and managing AI reels.

## Features

- Flask backend
- Responsive frontend using HTML and CSS
- Reel gallery
- Audio integration
- Template-based pages

## Tech Stack

- Python
- Flask
- HTML
- CSS

## Project Structure

```text
SAAS/
├── static/
├── templates/
├── main.py
└── README.md
```

## Installation

1. Clone the repository

```bash
git clone https://github.com/harshmaske65-lab/ai-reel-generator.git
```

2. Install Flask

```bash
pip install flask
```

3. Run the application

```bash
python main.py
```
